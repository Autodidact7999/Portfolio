My Projects


Covid-19 Norms Detection


In this project,I have develop System which can find out people who are not wearing a mask and Not maintain social Distancing. I proposed this whole System which can be integrated into the CCTV camera. In this project, I created a whole dataset trained Deep Learning Architecture on a dataset of people with and without a mask with accuracy of 99.8%. For Social Distancing, I have used YOLOv3 with DBSCAN clustering to identify potential invader.

Human Activity Recognition using Smartphone sensors

In this project, we have made an android Application. Which can predict whether the user is standing, walking, downstairs, Upstairs, Biking using Sensors in smartphones. We trained a model using Tensorflow and Keras and we integrate the model into the Android application.

Face recognization based attendance System

In this Attendance System, the attendance for students is marked using Face verification. The Faculty has the permission to take Attendance, add a student, modify student details. The attendance of a student recorded in Excel file including Multiparameter Search, by specifying the student ID, date of attendance, a period of Attendance. Tkinter was used for the development of the whole app. OpenCV and face_recognition API's were used for the development of Face Recognizer
